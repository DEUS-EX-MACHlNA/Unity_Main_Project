# 게임 시작 씬 생성 가이드

## 완료된 작업
✅ `TitleManager.cs` 스크립트 생성 완료 (`Assets/Scripts/Ryu/Title/TitleManager.cs`)

## Unity 씬 생성 (Unity MCP 또는 수동)

### 방법 1: Unity MCP 사용 (권장)
Unity MCP 서버가 켜져 있다면 다음 명령으로 씬을 생성할 수 있습니다:

1. **새 씬 생성**
   - 씬 이름: `Title`
   - 경로: `Assets/Scenes/Ryu/Title.unity`

2. **씬 구성 요소 생성** (Unity MCP 명령 사용)
   - Canvas 생성
   - EventSystem 생성
   - TitleText (TextMeshPro) 생성
   - StartButton (Button) 생성
   - FadeManager (SceneFadeManager) 생성
   - TitleManager (TitleManager) 생성

### 방법 2: Unity 에디터에서 수동 생성

#### Step 1: 새 씬 생성
1. Unity 에디터에서 `File → New Scene`
2. `Basic (Built-in)` 선택
3. `File → Save As`
4. 경로: `Assets/Scenes/Ryu/Title.unity`
5. 저장

#### Step 2: Canvas 생성
1. Hierarchy에서 우클릭 → `UI → Canvas`
2. Canvas 설정:
   - Render Mode: `Screen Space - Overlay`
   - Canvas Scaler 추가:
     - UI Scale Mode: `Scale With Screen Size`
     - Reference Resolution: `1920 x 1080`
     - Match: `0.5`

#### Step 3: EventSystem 생성
1. Canvas 생성 시 자동 생성됨
2. 없으면 Hierarchy에서 우클릭 → `UI → Event System`

#### Step 4: TitleText 생성
1. Canvas 하위에서 우클릭 → `UI → Text - TextMeshPro`
2. 이름 변경: `TitleText`
3. 설정:
   - Text: `게임 제목` (또는 원하는 제목)
   - Font Size: `72`
   - Alignment: `Center`, `Middle`
   - Color: 원하는 색상 (예: 흰색)
   - RectTransform:
     - Anchor: `Top Center`
     - Pos Y: `-100` (상단에서 100px 아래)

#### Step 5: StartButton 생성
1. Canvas 하위에서 우클릭 → `UI → Button - TextMeshPro`
2. 이름 변경: `StartButton`
3. Button 설정:
   - Interactable: `true`
   - Transition: 원하는 스타일 (예: Color Tint)
4. 하위 TextMeshPro 설정:
   - Text: `게임 시작` 또는 `START`
   - Font Size: `48`
   - Alignment: `Center`, `Middle`
5. RectTransform:
   - Anchor: `Bottom Center`
   - Pos Y: `100` (하단에서 100px 위)

#### Step 6: FadeManager 생성
1. Hierarchy에서 우클릭 → `Create Empty`
2. 이름 변경: `FadeManager`
3. `SceneFadeManager` 컴포넌트 추가
4. Canvas 하위에 FadeImage 생성:
   - Canvas 하위에서 우클릭 → `UI → Image`
   - 이름 변경: `FadeImage`
   - Color: `검은색 (0, 0, 0, 255)`
   - RectTransform:
     - Anchor: `Stretch Stretch`
     - Left, Right, Top, Bottom: 모두 `0`
   - Image 컴포넌트:
     - Color: `검은색 (R:0, G:0, B:0, A:0)` (초기 투명)

#### Step 7: TitleManager 생성
1. Hierarchy에서 우클릭 → `Create Empty`
2. 이름 변경: `TitleManager`
3. `TitleManager` 컴포넌트 추가

#### Step 8: Build Settings에 씬 추가
1. `File → Build Settings`
2. `Add Open Scenes` 클릭
3. Title 씬을 드래그하여 **첫 번째 위치**로 이동 (게임 시작 시 자동 로드)

## 최종 씬 구조

```
Title (Scene)
├── Canvas
│   ├── TitleText (TextMeshPro)
│   ├── StartButton (Button)
│   │   └── Text (TextMeshPro) - "게임 시작"
│   └── FadeImage (Image) - 검은색, 초기 투명
├── EventSystem
├── FadeManager (GameObject)
│   └── SceneFadeManager (Component)
└── TitleManager (GameObject)
    └── TitleManager (Component)
```

## 테스트 체크리스트

- [ ] 게임 실행 시 Title 씬이 먼저 로드되는지 확인
- [ ] 제목 텍스트가 화면에 표시되는지 확인
- [ ] 게임 시작 버튼이 화면에 표시되는지 확인
- [ ] 게임 시작 버튼 클릭 시 PlayersRoom 씬으로 전환되는지 확인
- [ ] 페이드 효과가 정상 작동하는지 확인
- [ ] 콘솔에 에러가 없는지 확인

## 문제 해결

### 버튼이 작동하지 않는 경우
- EventSystem이 씬에 있는지 확인
- StartButton의 Interactable이 true인지 확인
- TitleManager가 씬에 있는지 확인

### 페이드 효과가 작동하지 않는 경우
- FadeImage가 Canvas 하위에 있는지 확인
- FadeImage의 이름이 정확히 "FadeImage" 또는 "FadePanel"인지 확인
- SceneFadeManager 컴포넌트가 FadeManager에 추가되어 있는지 확인

### 씬 전환이 되지 않는 경우
- Build Settings에 Title 씬과 PlayersRoom 씬이 모두 추가되어 있는지 확인
- TitleManager의 콘솔 로그 확인
- SceneFadeManager가 정상적으로 찾아지는지 확인
