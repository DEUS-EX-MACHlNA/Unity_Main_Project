# 유저 플로우 다이어그램

이 문서는 게임의 전체 유저 플로우를 ASCII 다이어그램 형식으로 정리합니다.

---

```ascii
╔═══════════════════════════════════════════════════════════════════════════════╗
║                            게임 전체 유저 플로우                               ║
╚═══════════════════════════════════════════════════════════════════════════════╝

                    [게임 시작]
                         │
                         ▼
        ┌────────────────────────────────┐
        │      주요 행동 리스트           │
        │  - NPC와 상호작용              │
        │  - 아이템 탐색/사용            │
        │  - 장소 이동                   │
        └────────────────────────────────┘
                         │
                         ▼
        ╔═══════════════════════════════════════════════════════════╗
        ║              게임 상태 변화 (Game State Change)           ║
        ╚═══════════════════════════════════════════════════════════╝
                         │
        ┌────────────────┴────────────────┐
        │                                 │
        ▼                                 ▼
┌──────────────┐                  ┌──────────────┐
│   낮 시간    │◄─────────────────►│   밤 시간    │
│  (Day Time)  │                  │ (Night Time) │
└──────────────┘                  └──────────────┘
        │                                 │
        │                                 │
        │  프론트: 사용자 자유 서술       │  프론트: 없음
        │  백엔드: 게임 상태 변화 및      │  백엔드: 게임 상태 변화 및
        │         소설 응답              │         소설 응답
        │                                 │
        │                                 │
        └────────────────┬────────────────┘
                         │
                         ▼
        ╔═══════════════════════════════════════════════════════════╗
        ║                        백엔드 (Backend)                   ║
        ║  - 게임 대사 (Game Dialogue)                              ║
        ║  - 게임 상태 변화 (Game State Change)                     ║
        ║  - 소설 응답 (Novel Response)                            ║
        ╚═══════════════════════════════════════════════════════════╝
                         │
                         │
        ┌────────────────┴────────────────┐
        │                                 │
        ▼                                 ▼
  [낮 시간 응답]                    [밤 시간 응답]
  - narrative                        - dialogues[]
  - state_result                     - state_result
  - npc_stats                        - npc_disabled_states
  - inventory_add/remove             - flags
  - flags                            - ending_info
  - ending_info


╔═══════════════════════════════════════════════════════════════════════════════╗
║                          상세 플로우 설명                                      ║
╚═══════════════════════════════════════════════════════════════════════════════╝

┌─────────────────────────────────────────────────────────────────────────────┐
│ 낮 시간 (Day Time) 플로우:                                                  │
│                                                                             │
│  1. 사용자가 자유롭게 텍스트 입력 (자유 서술)                                │
│  2. 프론트엔드가 백엔드 API 호출 (POST /api/v1/game/{gameId}/step)          │
│  3. 백엔드가 게임 상태를 판단하고 응답 반환:                                 │
│     - narrative: 게임 내 표시할 대화/서술 텍스트                            │
│     - state_result: 게임 상태 변화 (NPC 통계, 아이템, 플래그 등)            │
│     - ending_info: 엔딩 트리거 정보 (선택적)                                │
│  4. 턴 소모 (remainingTurns 감소)                                          │
│  5. 턴이 0이 되면 밤 시간으로 전환                                          │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 밤 시간 (Night Time) 플로우:                                                │
│                                                                             │
│  1. 턴이 0이 되면 자동으로 Night 씬으로 전환                                │
│  2. 프론트엔드가 백엔드 API 호출 (POST /api/v1/game/{gameId}/night_dialogue)│
│  3. 백엔드가 밤 대화 응답 반환:                                             │
│     - dialogues[]: 가족 간 대화 배열 (순서대로 표시)                        │
│     - state_result: 게임 상태 변화 (NPC 무력화, 플래그 등)                  │
│     - ending_info: 엔딩 트리거 정보 (선택적)                                │
│  4. 대화 완료 후 다음 날로 진행 (AdvanceToNextDay)                          │
│  5. 인간성 10% 감소 (시간 경과 페널티)                                      │
│  6. 새로운 낮 시간 시작 (턴 리셋)                                            │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 게임 상태 변화 (Game State Change):                                         │
│                                                                             │
│  - NPC 통계: trust, suspicion, fear                                        │
│  - 아이템: inventory_add, inventory_remove                                 │
│  - NPC 무력화: npc_disabled_states (remaining_turns)                       │
│  - 플래그: flags (시나리오/엔딩 분기용)                                     │
│  - 인간성: humanity (0~100)                                                 │
│  - 엔딩: ending_info (ending_id, name, epilogue_prompt)                    │
└─────────────────────────────────────────────────────────────────────────────┘
```
